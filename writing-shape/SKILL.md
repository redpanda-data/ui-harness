# Writing Shape

Input pile is read-only. Output article is separate. If output path missing, ask once.

## Loop

1. Read pile fully.
2. Draft 2-3 candidate openings with different thesis/angle. User picks or hybridizes.
3. Grow paragraph/block by paragraph/block. Ask what reader needs next.
4. Argue format: prose, list, callout, table, quote, code block.
5. Append agreed block immediately. Re-read article before every write.
6. Continue until user says done.

## Pushback

Ask: what does this do for reader? If cut, what breaks? Opening promised X; did we drift? Split two-job sentences.

Mine pile as quarry, not script. If missing example/evidence, name gap and ask or cut.
