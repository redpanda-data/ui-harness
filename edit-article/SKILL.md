# Edit Article

1. Split article by headings. Infer main point per section.
2. Check dependency order: concepts appear before later sections rely on them.
3. Confirm section order with user.
4. Rewrite section-by-section: clearer, tighter, better flow. Max 240 chars per paragraph unless user asks otherwise.
5. Preserve thesis, facts, examples, voice. Cut repetition/filler.
